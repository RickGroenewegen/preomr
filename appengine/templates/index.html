Abstract

We provide an introduction to Optical Music Recognition research along with a tool for doing pre-processing of images to aid current commerical tools when the purpose is automated gathering of statistical information from large music corpora. This is implemented by using the Gamera Open Source framework for Document Image Analysis and uses both a K-Nearest-Neighbor recognizer with rejection based on confidence measures and different image projection and estimation techniques to remove text and dynamics from a high quality image of sheet music.

Experiments show 99% precison for text and dynamics with a recall from 84% to 98% for the tool but it was not possible to verify benificial effect of the PreOmr processing.
